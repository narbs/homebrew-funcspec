# homebrew-funcspec

Homebrew tap for the [funcspec CLI](https://github.com/narbs/funcspec-cli).

## Install

```sh
brew install narbs/funcspec/funcspec
```

## Update

```sh
brew update
brew upgrade funcspec
```

The formula in this repo is regenerated automatically by the funcspec-cli release workflow on each new tag. Manual edits will be overwritten.
